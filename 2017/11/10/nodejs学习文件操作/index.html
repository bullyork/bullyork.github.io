File System

NodeJS通过fs内置模块提供对文件的操作。fs模块提供的API基本上可以分为以下三类：

  • 文件属性读写：fs.stat、fs.chmod、fs.chown等
  • 文件内容读写： fs.readFile、fs.readdir、fs.writeFile、fs.mkdir等等
  • 底层文件操作：fs.open、fs.read、fs.write、fs.close等等
